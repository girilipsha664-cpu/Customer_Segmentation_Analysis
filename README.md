# Customer Segmentation Analysis

## Internship Project

### Project Overview
This project performs customer segmentation analysis using machine learning techniques to group customers based on purchasing behavior and income patterns.

### Objectives
- Perform data cleaning and preprocessing
- Analyze customer purchasing behavior
- Apply K-Means clustering
- Visualize customer segments
- Generate business insights and recommendations

### Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

### Dataset
ifood_df.csv

### Key Features
- Data Cleaning
- Exploratory Data Analysis
- Feature Scaling
- K-Means Clustering
- Elbow Method
- Customer Segmentation Visualization
- Cluster Analysis

### Business Insights
The analysis identified multiple customer segments based on income and purchasing behavior. These insights can help businesses improve targeted marketing and customer retention strategies.

### Conclusion
The project successfully demonstrated customer segmentation using machine learning and data visualization techniques with Python.
